# Network Portfolio Labs (Packet Tracer)

HQ–Branch–ISP 구조를 기반으로, 라우팅/이중화/보안/운영 요소를 단계적으로 확장한 네트워크 실습.
각 LAB은 “목표 → 설계 의도 → 구성 → 검증(증빙) → 제약” 흐름으로 문서화.

## 기술 구성
- Routing: OSPF, Default propagation, Static route
- Campus: VLAN, Inter-VLAN Routing, STP, EtherChannel
- Redundancy: HSRP, STP Root design
- Security: ACL, Port-Security (DHCP Snooping/DAI 등은 해당 LAB 포함 시)
- Operations: Syslog, NTP, SSH (추가 예정)

## Labs
- **LAB1 – OSPF / NAT(PAT·Static) / DHCP Relay 기본 구성**
- **LAB2 – VLAN / Inter-Routing**
- **LAB3 – LAB1 + LAB2 / DNS**
- **LAB4 – HSRP / STP**
- **LAB5 – ACL / Port-Security**
- **LAB6 – NTP / Syslog**
- **LAB7 – SSH (진행중)**
