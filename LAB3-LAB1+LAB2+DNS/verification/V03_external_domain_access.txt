[검증항목]
- 항목: 내부 PC에서 외부 서버(8.8.8.8, 외부 Web Server)에 대해 Domain Name 기반 접근 가능 여부 확인
- 목적: DNS를 통해 외부 자원도 도메인으로 해석/접근 가능한지 확인하고, 외부 구간 NAT까지 포함해 정상 동작하는지 확인
- 기대결과:
  - 도메인 해석 성공(외부 IP 반환)
  - 8.8.8.8(Internet loopback) 및 외부 웹 서버 접속/통신 성공

[구성 참고]
- Internet(loopback 8.8.8.8) 존재 
- 외부 서버망(203.0.20.0/30) 인터페이스 구성 
- ASA에서 내부→외부 PAT 수행 

[사전조건]
- DNS 서버에 외부 웹 서버 도메인 A 레코드 등록(web.ext.com → 203.0.20.2)

[실행]
1) 내부 PC에서 외부 도메인 해석 확인
- nslookup <internet도메인 또는 ext-web 도메인>

2) 통신 확인
- ping 8.8.8.8
- (외부 웹) Web Browser에서 http://<web.ext.com> 접속
