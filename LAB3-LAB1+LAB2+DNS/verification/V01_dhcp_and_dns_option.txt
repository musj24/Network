[검증항목]
- 항목: 내부 PC가 DHCP를 통해 IP 및 DNS 서버 정보를 정상적으로 할당받는지 확인
- 목적: 중앙 DHCP 서버(192.168.50.3)를 통해 각 VLAN PC가 IP/GW/DNS를 정상 수신하는지 확인
- 기대결과:
  - HQ VLAN10: 192.168.1.x / GW 192.168.1.1 / DNS 192.168.50.3
  - HQ VLAN20: 192.168.2.x / GW 192.168.2.1 / DNS 192.168.50.3
  - HQ VLAN30: 192.168.3.x / GW 192.168.3.1 / DNS 192.168.50.3
  - Branch VLAN10: 192.168.10.x / GW 192.168.10.1 / DNS 192.168.50.3
  - Branch VLAN20: 192.168.20.x / GW 192.168.20.1 / DNS 192.168.50.3
  - Branch VLAN30: 192.168.30.x / GW 192.168.30.1 / DNS 192.168.50.3

[구성 참고]
- HQ ROAS 서브인터페이스에 helper 적용
- Branch-SW SVI에 helper 적용

[실행]
1) 각 VLAN PC에서 DHCP 할당 상태 확인(PT PC 설정 화면 또는 ipconfig)
- HQ VLAN10/20/30 PC 1대씩
- Branch VLAN10/20/30 PC 1대씩
