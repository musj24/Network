[검증항목]
- 항목: 내부 PC에서 내부 서버에 대해 Domain Name 기반 접근 가능 여부 확인
- 목적: DNS 서버(192.168.50.3)에 등록된 내부 서버 도메인(A 레코드)을 사용하여 접근 가능한지 확인
- 기대결과:
  - 내부 PC에서 "내부 웹/내부 서버 도메인"으로 접속/해석 성공
  - nslookup 결과가 내부 서버 IP로 반환

[사전조건]
- DNS 서버에 내부 서버 A 레코드 등록(web.intra.local → 192.168.50.2)
- DHCP로 DNS(192.168.50.3)가 클라이언트에 전달됨(V01과 연계)

[실행]
1) 내부 PC에서 도메인 해석 확인
- PC: HQ 또는 Branch VLAN PC
- 명령/동작: nslookup <내부서버도메인>

2) 내부 PC에서 내부 서버 접근(웹이면 HTTP)
- PC: HQ 또는 Branch VLAN PC
- 동작: Web Browser에서 http://<web.intra.local> 접속
