항목: Syslog 서버에서 Distribution/HQ 장비 이벤트 로그 수신 확인

목적: 중앙 Syslog 서버(192.168.50.4)로 로그가 수집되어 운영 가시성이 확보되는지 확인

기대결과:

Syslog 서버에서 DSW/HQ 장비가 전송한 로그가 수신됨

장비 hostname/메시지(링크 업/다운, 라우팅 이벤트 등)가 기록됨

[구성 참고]

Syslog Server: 192.168.50.4

각 장비에 logging host 192.168.50.4 구성

PT 제약: Access(L2) 스위치 이벤트가 서버로 전달되지 않는 경우가 있어, DSW/HQ 이벤트 중심으로 검증

[실행]

1. Syslog 서버 서비스/로그 수신 화면 확인

장비: Syslog Server(192.168.50.4)

방법: PT Server -> Services -> SYSLOG(활성/로그 수신 화면 확인)

2. DSW/HQ에서 로깅 대상 설정 확인

장비: DSW1/DSW2/HQ1/HQ2

기대: 로컬 로그가 생성되고, 설정된 logging host가 존재
